Five deployment levels from cost-optimized POC to fully supported Enterprise Production.

All pricing is based on Azure public pricing and includes Infrastructure as Code (IaC) deployment strategies for customer Azure tenants.

Current Deployment: Staging POC with ACA (scale-to-zero) and PostgreSQL B1ms SKU (~$23/month idle, ~$50-80/month light usage).

üí° FinOps Optimization

Engram is designed with a FinOps-first approach. Key cost optimizations include:

  • Scale-to-zero on Azure Container Apps for idle workloads
  • Reserved instances for predictable production workloads
  • Spot instances for batch/background processing
  • Tiered storage with automatic lifecycle policies
  • Right-sizing recommendations via Azure Advisor
