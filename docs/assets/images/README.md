## Generating Images

Use the JSON prompts in [visual-development.md](../../visual-development.md) with Nano Banana Pro or similar AI image generation tool.

### Color Palette

Maintain consistency with these colors:

| Name | Hex | Usage |
|------|-----|-------|
| Background | `#0a0e1a` | Dark background |
| Primary (Cyan) | `#00d4ff` | Elena, primary actions |
| Secondary (Purple) | `#a855f7` | Marcus, secondary elements |
| Accent (Green) | `#10b981` | Success, highlights |
| Text | `#e2e8f0` | Primary text |
| Text Dim | `#94a3b8` | Secondary text |

### Style Guidelines

1. **Dark theme** - All visuals should use dark backgrounds
2. **Glassmorphism** - Use subtle transparency and blur effects
3. **Glow effects** - Add subtle glows for interactive elements
4. **Consistent typography** - Use clean, modern sans-serif fonts
