# Engram Visual Assets

This directory contains the visual assets for the Engram platform wiki.

## Generated Images (Nano Banana Pro)

The following professional images have been generated:

| Filename | Description | Status |
|----------|-------------|--------|
| `engram-logo.png` | Neural network "E" logo with cyan-purple gradient | ✅ Generated |
| `voice-interaction-flow.png` | Speech-to-Avatar pipeline with latency breakdown | ✅ Generated |

| `temporal-workflow.png` | Agent execution state diagram with Temporal | ✅ Generated |
| `elena-portrait.png` | Dr. Elena Vasquez professional headshot | ✅ Generated |
| `marcus-portrait.png` | Marcus Chen professional headshot | ✅ Generated |

## Adding Images

Save the generated images to this directory with the filenames listed above.

```bash
# Example: Copy images from Downloads
cp ~/Downloads/engram-logo.png ./
cp ~/Downloads/voice-interaction-flow.png ./
cp ~/Downloads/engram-platform-architecture.png ./
cp ~/Downloads/temporal-workflow.png ./
cp ~/Downloads/elena-portrait.png ./
cp ~/Downloads/marcus-portrait.png ./
```
